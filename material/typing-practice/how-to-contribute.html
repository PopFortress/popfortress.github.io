在 GitHub 仓库中贡献新文章

感谢您对本站和“打字练习 | Just Enough Typewriting”的支持和对开源事业的奉献。

为了向我们提交您的文章，请参考以下步骤：

  1. 在 GitHub 上 Fork 本仓库。
  2. 在您的 Fork 中创建一个新的分支。
  3. 在 /material/typing-practice/articles/ 目录下的 index.json 文件中的对应语言分类中添加您的文章名称。
  4. 在 articles 目录下与您文章语言对应的目录中创建文本文件，命名格式如 <文章名>.txt，使 <文章名> 与上一步骤中您添加的名称对应。
  5. 在此文件中编写您的文章，请注意一行文本不宜超过 50 字或 100 个英文单词。
  6. 提交您的更改，并创建一个 Pull Request。

我们将在 3-4 个工作日内审核您的更改，审核通过后您的更改将被合并，使您提交的文章成为预设的一部分。
