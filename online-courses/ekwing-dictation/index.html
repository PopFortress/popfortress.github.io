下载适用于 Python 的程序源文件。

ekwing-auto-dictation.zip

前往 OneDrive 查看使用文档 * 受限制地区用户可能无法访问此链接。

1. 前往 OneDrive 下载程序包。

2. 检查是否安装 Python（版本 ≥ 3.10）以及 keyboard 库。

若未安装 Python 请前往 Python 官网下载最新版本进行安装。

若未安装 keyboard 库请通过以下命令来安装。

pip install keyboard

3. 打开翼课网网页端，在打开词语听写窗口前打开 F12 开发人员工具。打开听写窗口后找到并复制名为 proxy 请求的响应结果。

4. 解压压缩文件包，将结果粘贴至 vocabulary 目录下的 words.json 文件中。

5. 在压缩包根目录下打开终端，运行命令：

python .\main.py

6. 按换行键确定开始后在 3 秒内切换到翼课网网页端，点击开始听写按钮。

7. 享受自动听写带来的便捷！

若遇任何问题，请自行搜索解决。
